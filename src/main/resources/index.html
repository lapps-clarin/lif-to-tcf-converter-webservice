Demo Service

There is an example of web service included in this demo.

A sample LIF(LAPPS Grid interchange format) to TCF(WebLicht interchange format) converter

This web service is an example of a simple converter that converts a LIF file to TCF.

This web service demonstrates the case when the underlying processing tool object is not expensive to create and it does not consume a lot of memory. In such a case it is convenient to create the tool object with each client POST request, as shown in the service implementation. You don't need to worry about whether the tool is thread-safe or not.

You can test the service endpoints using curl or wget as follows:

  • download the file inputMulti_con.lif file and run the following command
    • wget --post-file=inputMulti_con.lif --header='Content-Type: application/json' con/stream

  • more examples of the converters:

  • download the file inputTok_con.lif file. This is an example of converting token layer.
    wget --post-file=inputTok_con.lif --header='Content-Type: application/json' con/stream
  • download the file inputPos_con.lif file. This is an example of converting Pos Tagger layer.
    wget --post-file=inputPos_con.lif --header='Content-Type: application/json' con/stream
  • download the file inputNer_con.lif file. This is an example of converting Name Entity Recognition layer.
    wget --post-file=inputNer_con.lif --header='Content-Type: application/json' con/stream
  • download the file inputCon_con.lif file. This is an example of converting constituent layer.
    wget --post-file=inputCon_con.lif --header='Content-Type: application/json' con/stream
  • download the file inputDep_con.lif file. This is an example of converting dependency layer.
    wget --post-file=inputDep_con.lif --header='Content-Type: application/json' con/stream
